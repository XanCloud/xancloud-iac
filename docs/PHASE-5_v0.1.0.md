# Fase 5 — Producto

> **Duración estimada:** 4-6 semanas | **Dependencia:** Fase 3 completa (al menos 2 blueprints) | **Estado:** No iniciada

## Objetivo

Transformar el acelerador de consultoría en un producto vendible con documentación de nivel producto, registry privado, demos y onboarding automatizado.

## Entregables

- MkDocs site — Documentación navegable con guías de adopción, diagramas por escenario, troubleshooting
- OCI Registry — Módulos publicados en ECR con versionamiento semántico y changelogs
- Template `client-scaffold/` — Script que genera un repo completo para un nuevo cliente con backend, environments y pipeline pre-configurados
- Demo environment — Landing zone desplegada que se pueda mostrar a prospects
- Onboarding automatizado — Guía paso a paso + script de setup inicial

## Criterio de completitud

- [ ] MkDocs site desplegado y navegable
- [ ] Al menos 5 módulos publicados en OCI registry con tags semánticos
- [ ] Client scaffold genera un repo funcional en < 5 min
- [ ] Demo environment desplegada y accesible
- [ ] Primer cliente (real o piloto) onboarded con el producto
